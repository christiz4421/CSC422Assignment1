Description
This program implements a simple pet database that allows users to add, view, search, delete, and update pet records. It was developed for CSC 422 Assignment 1 and expanded across three releases.

Features
Release 1
• 	Add pets with name and age
• 	Display all pets in a formatted table
• 	Show total number of pets
Release 2
• 	Search pets by name or age
Release 3
• 	Remove a pet by ID
• 	Update an existing pet’s name or age
