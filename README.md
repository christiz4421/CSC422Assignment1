## Description
This program creates a simple pet database that allows users to add pets and view them in a
formatted table. It is the first release of the Pet Database Program, developed for CSC 422 
Assignment 1 Part 2.

## Features
- Add pets with name and age
- Display all pets in a table format
- Show total number of pets in the database

## Files Included
- `Pet.java` – defines the Pet class
- `PetDatabaseProgram.java` – main program logic and user interface
- `CSC422Assignment1.iml` – IntelliJ project file 

## How to Run
1. Open the project in IntelliJ IDEA or any Java IDE
2. Compile and run `PetDatabaseProgram.java`
3. Follow the menu prompts to add and view pets
