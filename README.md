Description
This program implements a simple pet database that allows users to add, view, search, delete, and update pet records. It was developed for CSC 422 Assignment 1 (Releases 1-3) and expanded in Assignment 2 to include file loading and saving and error handling.

Features
Release 1
• 	Add pets with name and age
• 	Display all pets in a formatted table
• 	Show total number of pets
Release 2
• 	Search pets by name or age
Release 3
• 	Remove a pet by ID
• 	Update an existing pet’s name or age 

Assignment 2 Additions

Milestone 3 File Loading/Saving
- Load pets from a text file when the program starts
- Save pets to a text file when the program exits

Milestone 4 Error Handling
- Limit database to 5 pets
- Validate pet age range
- Validate input format
- Validate ID input
- Validate menu choices

How to Run
1. 	Open the project in IntelliJ IDEA or any Java IDE
2. 	Compile and run 
3. 	Use the menu to add, view, search, delete, and update pets
4. 	Pets are automatically loaded at startup and saved at exit
